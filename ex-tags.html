Surgimento da Internet

A internet nasceu durante a Guerra Fria como ARPANET, uma rede descentralizada para fins militares. No entanto, o conceito de web (páginas interligadas) surgiu em 1989 no CERN, pelas mãos de Tim Berners-Lee. Ele desenvolveu o HTML para permitir que pesquisadores pudessem ler documentos e "pular" para outros através de hiperlinks.

Como funciona o HTML

O HTML não é programação, mas uma linguagem de marcação. O navegador (Chrome, Firefox, Edge) lê o arquivo de texto e, ao encontrar as marcações, desenha o layout na tela. Ele organiza o conteúdo em uma estrutura lógica onde cada elemento possui um papel específico na hierarquia do documento.

O que são Tags?

Tags são os comandos fundamentais do HTML. Elas funcionam como etiquetas que "envelopam" o conteúdo. A anatomia básica de uma tag consiste em:

  1. Abertura: <tag>
  2. Conteúdo: O texto ou elemento interno.
  3. Fechamento: </tag>

Existem também as tags vazias, que não possuem conteúdo interno e se fecham sozinhas, como a tag de imagem ou de quebra de linha.

Principais Tags de Semântica

Tags semânticas descrevem o significado do seu conteúdo. Abaixo, uma tabela explicativa:

Tag Descrição Semântica
<header> Define o cabeçalho da página ou de uma seção específica.
<main> Representa o conteúdo principal e central do documento.
<article> Define um conteúdo independente e autossuficiente (como um post).
<section> Agrupa conteúdos que possuem uma temática relacionada.
<footer> Contém informações de rodapé, como autor e contato.

Principais Tags de Mídia

As tags de mídia permitem que a web deixe de ser apenas texto e se torne visual e auditiva:

  • <img>: Insere imagens (requer o atributo src e alt).
  • <video>: Incorpora players de vídeo nativos.
  • <audio>: Incorpora reprodução de som.
  • <canvas>: Utilizada para desenhar gráficos e animações via script.

Principais Tags de Responsividade e Acessibilidade

Para garantir que o site funcione em qualquer dispositivo e para qualquer pessoa:

  • Viewport: A meta tag viewport ajusta a escala do site em telas pequenas.
  • Alt Text: O atributo alt em imagens é crucial para leitores de tela usados por pessoas com deficiência visual.
  • Labels: A tag <label> conecta textos a campos de formulário, melhorando a área de clique e a leitura assistiva.
